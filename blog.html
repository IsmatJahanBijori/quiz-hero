1. Difference between map and forEach, filter and find?

The first difference between map and forEach is about to returning value. map() returns a new array with updated value but forEach() returns no value as you can say undefined. The differnece between filter and find is also about to returning value. find() returns only first value of an array but filter() returns containing all element in an array.

2. Difference between local storage, session storage and cookies?

Local Storage: It helps to store and retrieve data from browser. Saved infromation will never be cleared unless the code is deleted.
Session Storage: It helps to store data on the browser depending on the memory. Saved infromation will be deleted when the page session end or the tab is closed.
Cookies: It is a piece of personal information of users that is saved in a web browser so that a website can access it later.

3. What is event loop and how does it work?

The Event Loop is to keep track of the call stack and the callback queue. If the call stack is empty, the Event Loop will take the first event from the queue and will push it to the call stack. Making a request to an internal or external "event provider" and then calling the appropriate event handler is how the event loop operates.
