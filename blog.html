1. Difference between map and forEach, filter and find?

The first difference between map and forEach is about to returning value. map() returns a new array with updated value but forEach() returns no value as you can say undefined. The differnece between filter and find is also about to returning value. find() returns only first value of an array but filter() returns containing all element in an array.
